You will need to download R (latest version for now R-4.1.1) and place it on your C drive at C:\R\R-4.1.1
Make sure you have added the librairies Hmisc, XLConnect, Haven, SASxport and utils to you R library.

From R console:

install.packages ( "Hmisc" )
install.packages ( "XLConnect" )
install.packages ( "haven" )
install.packages ( "SASxport" )
install.packages ( "R.utils" )
